Type Description
IEnumerable<T>
Remarks

This enumerable collection contains items whose the count is 0.

Enumerators retured by this enumerable collection cannot be used to modify the Counter<T>. For example, the following enumeration raises an InvalidOperationException.

var counter = new Counter<int>();
counter.Add(new[] { 1, 2, 3 });
foreach (int item in counter.Items)
{
  counter.Reset(item);
}
