Remarks

This enumerable collection also contains items whose the count is 0.

The enumerator retured by this enumerable collection cannot be used to modify the Counter<T>. For example, the following enumeration raises an InvalidOperationException.

var counter = new Counter<int>();
counter.AddRange([1, 2, 3]);
foreach (int item in counter.UniqueItems)
  counter.Reset(item);
